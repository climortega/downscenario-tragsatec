MÉTODOS

Para la obtuvieron se consideraron las observaciones de la red de AEMET y diferentes productos de reanálisis (ERA5, ERA5-Land, E-OBS, CERRA, CERRA-Land y COSMO-REA6) y satelitales (CLARA-A3) con el objetivo de encontrar el reanálisis más cercano a las observaciones meteorológicas disponibles. Para más información, ver la sección de Datos.

Cada variable tiene una serie temporal observacional con diferente completitud y consistencia. Por tanto, primero se llevó a cabo un análisis de sensibilidad en el que, para cada variable, se seleccionara la mínima completitud de las estaciones (siempre mayor que el 50% de los datos) que, sobre un periodo determinado, resultara en el mayor número de estaciones disponibles. Este análisis se llevó a cabo para un periodo posterior a 1995 (el factor limitante de COSMO-REA6).

Para llevar a cabo la evaluación, todas las bases de datos se compararon con la red observacional interna de AEMET, teniendo en cuenta que CERRA-Land no cuenta con datos disponibles para la velocidad del viento y la humedad relativa y que solo se utilizaron CERRA, CERRA-Land y CLARA-A3 para la radiación. Esta comparación se realizó a través del software de Python pyGridEval, que devuelve un extenso número de métricas estadísticas que permiten comparar las bases de datos. pyGridEval (Carlos Correa) se encuentra disponible en el siguiente link.

La siguiente lista describe las métricas estudiadas:

  • Correlación. Medida que indica la relación lineal entre dos variables, siendo 1 el valor que muestra una relación positiva perfecta.

  • Prueba K-S (Kolmogorov-Snirnov). Prueba estadística que comparar dos distribuciones. El estadística es la máxima diferencia entre las funciones de distribución acumulada. El valor p es la probabilidad de observar esas diferencias si ambas distribuciones fueran iguales, siendo una p pequeña el indicador de que las distribuciones son distintas.

  • Curtosis. Mide la forma de su distribución de frecuencias. Una curtosis grande implica un mayor número de valores de la variable muy dispersos y muy lejos del centro de la misma.

  • Error absoluto medio.

  • Percentiles 10, 90 y 95.

  • Raíz del error cuadrático medio. Penaliza los errores más grandes.

  • Asimetría estadística de la distribución.

  • Desviación estándar (std). Mide la dispersión del conjunto de datos.

  • Varianza. Similar a la desviación estándar, pero en escala cuadrática.

  • Distancia de Wasserstein. Mide la distancia entre dos distribuciones como el "coste" de transformar una en otra. Muy útil cuando las distribuciones no son gaussianas, pues es sensible a la forma, la posición y la dispersión.

  • Número de días húmedos. Solo se calcula para la precipitación.

  • Métrica de rendimiento SEEPS (Stable Equitable Error in Probability Space). Evalúa predicciones categóricas, tiene en cuenta la climatología local y penaliza errores según su gravedad. Un valor de 0 indica un desempeño perfecto. Solo se calcula para la precipitación.

  • Sesgo. Diferencia entre la medida del modelo y la observada. Se calcula para diferentes métricas.

  • Sesgo relativo. Sesgo relativo respecto al valor observado, lo cual permite comparar errores en distintas escalas. Solo se calcula para la precipitación.

  • Sesgo multiplicativo. Indica si el modelo no escala adecuadamente los valores. Solo se calcula para la precipitación.
