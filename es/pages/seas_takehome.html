CONCLUSIONES

  • El método de regionalización DeepESD se ha aplicado satisfactoriamente para generar pronósticos retrospectivos regionalizados utilizando el modelo de precipitación (DeepESDpr) para esta variable y el modelo de temperatura (DeepESDtas) para las demás.

  • Los pronósticos retrospectivos regionalizados han sido verificados mediante la evaluación del invierno extendido (NDJFM, agregaciones de 3 y 5 meses) desde 1997 hasta 2016.

  • La temperatura y el viento muestran buenos valores de correlación Spearman.

  • Se han obtenido prometedores indicadores de rendimiento AUC y RPS para la temperatura.

  • Se ha conseguido cierta capacidad predictiva para el viento y también, aunque menor, para la humedad relativa.

  • Las métricas de redimiento podrían mejorarse mediante la utilización de modelos específicos para cada variable.

  • Las diferencias entre agregaciones varían según la variable.

  • Estas conclusiones deberían considerarse al estudiar los pronósticos regionalizados tanto para 2024 como para 2025.

María Ortega, Irene Rodríguez-Muñoz, Jesús Gutiérrez-Fernández (Tragsatec) ● Esteban Rodríguez-Guisado (AEMET)
