METHODS

For the evaluation, observations from the AEMET network and different reanalysis products (ERA5, ERA5-Land, E-OBS, CERRA, CERRA-Land, and COSMO-REA6) and satellite data (CLARA-A3) were considered, with the aim of finding the reanalysis most similar to the available meteorological observations. For more information, see the Data section.

Each variable has observational time series of varying completeness and consistency. Therefore, a sensitivity analysis was first performed in which, for each variable, the minimum station completeness (always greater than 50%) that, over a given period, yields the largest number of available stations was selected. This analysis was performed for a period after 1995 (the limiting factor for COSMO-REA6).

Temperature 80% 1995-2018 974

To perform the evaluation, all datasets were compared with AEMET internal observation network, taking into account that wind speed and relative humidity data were not available from CERRA-Land and that only CERRA, CERRA-Land, and CLARA-A3 were used for radiation. This comparison was carried out using Python software pyGridEval, which returns a large number of statistical metrics that allow for comparison between datasets. You can access pyGridEval (Carlos Correa) in the following link.

The following list describes all studied metrics:

  • Correlation. Indicates the linear relationship between two variables, with 1 being the value that shows a perfect positive relationship.

  • K-S (Kolmogorov-Smirnov) test. A statistical test that compares two distributions. Stat is the maximum difference between cumulative distribution functions. p-value is the probability of observing these differences if both distributions were equal, with a small p-value indicating that distributions are different.

  • Kurtosis. Measures the shape of a frequency distribution. High kurtosis implies a greater number of values that are widely dispersed and far from its center.

  • Mean absolute error.

  • 10th, 90th, and 95th percentiles.

  • Root mean square error. Penalizes larger errors.

  • Statistical skewness of the distribution (skew).

  • Standard deviation (std). Measures the dispersion of the dataset.

  • Variance. Similar to the standard deviation, but on a quadratic scale.

  • Wasserstein distance. Measures the distance between two distributions as the "cost" of transforming one into the other. Very useful when distributions are not Gaussian, as it is sensitive to shape, position, and dispersion.

  • Number of wet days. Calculated only for precipitation.

  • SEEPS (Stable Equitable Error in Probability Space) Skill Score. Evaluates categorical predictions, takes local climatology into account, and penalizes errors according to their severity. A value of 0 indicates perfect performance. Calculated only for precipitation.

  • Bias. Difference between model measurement and observed measurement. Calculated for different metrics.

  • Relative bias. Bias relative to the observed value, which allows for comparing errors on different scales. Calculated only for precipitation.

  • Multiplicative bias. Indicates whether the model does not adequately scale values. It is only calculated for precipitation.
